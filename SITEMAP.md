## Navigation Structure

All pages include:
- Header with site title and tagline
- Full navigation menu
- Main content area
- Footer with copyright and links

## Dropdown Menu

The "Flock" menu item has a dropdown with:
- Current Sires
- Reference Sires  
- Ewes

## External Links

- Facebook: https://www.facebook.com/creekroadfarms/
- Conner Prairie: http://www.connerprairie.org/
- Livestock Conservancy: https://livestockconservancy.org/
- Email: nddcreek@gmail.com

## Content Organization

### Static Pages (No frequent updates)
- History
- Philosophy
- Flock (overview)

### Dynamic Pages (May need updates)
- Home (family updates)
- Current Sires (when changing rams)
- Ewes (as flock composition changes)
- Shows (show results and plans)
- Sales (availability of products)
- Contact (if contact info changes)

## Mobile Navigation

On mobile devices:
- Navigation menu stacks vertically
- Dropdown menu expands/collapses
- All content remains accessible
- Touch-friendly interface

## SEO Structure

Each page has:
- Unique title tag
- Meta description
- Semantic HTML structure
- Descriptive headings (H1, H2)
- Internal linking
- Mobile-friendly design

## Future Expansion

Easy to add:
- Photo galleries (add HTML pages)
- Individual sheep profiles (create new pages)
- Show results pages (yearly pages)
- Sale listings (seasonal updates)
- News/updates section (static pages)

Just create new HTML files following the template structure!
